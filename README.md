An integrated game information and newsletter platform, built with Next.js 14+ and Tailwind CSS v4.
Designed to offer PC and Console gamers a unified view of the gaming world.

## Features (Phase 1 MVP)

- **Home Dashboard**: Featured games, New Releases, Trending Top 10.
- **Platform Picks**: Filter games by PC, PlayStation, Xbox, Switch.
- **GameQuick Briefing**: Top headline news and countdowns.
- **Newsletter Subscription**: Join the daily briefing.
- **Design**: Premium "PlayStation-like" dark UI with glassmorphism.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Font**: Inter, JetBrains Mono
